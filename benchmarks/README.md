# Benchmarks — Reproducibility Proof Layer

This folder contains public verification artifacts demonstrating that the BRC kernel is:

- deterministic per seed
- variable across seeds
- ledger-verifiable
- cryptographically reproducible

## Structure

benchmarks/
├─ runs/
│  ├─ seedX_stepsY.jsonl
│  └─ seedX_stepsY.hash
│
└─ results/
   ├─ latest_test.json
   └─ multiseed_summary.json

### runs/

Archived ledgers from deterministic runs.
These are copies of ledger/runs/ exported for public verification.

Each pair:
- .jsonl → append-only ledger
- .hash → final chain hash

Anyone can verify with:

python -c "from brc.ledger.verify import verify_chain_hash; print(verify_chain_hash('file.jsonl','file.hash'))"

### results/

Summaries generated by tests and stress harness.

- latest_test.json → single-seed determinism proof
- multiseed_summary.json → multi-seed stability report

## Meaning

If:
- determinism_all = true
- ledger_all = true
- variation_ok = true

then the kernel satisfies reproducible execution invariants.
